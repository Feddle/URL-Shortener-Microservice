URL Shortener Microservice

Returns the given url shortened in a JSON response.

https://github.com/Feddle/URL-Shortener-Microservice

Example usage

https://short-url-feddle.glitch.me/new/https://www.google.com

Example output

{"original_url":"https://www.google.com","short_url":"https://short-url-feddle.glitch.me/3ea7c7"}
